ipm
=================

IP Messagener 簡易互換ライブラリ

主にメッセージの送受信のみを行う簡易互換ライブラリ

最初は通常のソケット通信、そのうちNIO2でデータ受信を実施する。
ライブラリにリスナーを登録することで、各イベントを受信できるようにする

利用方法
サンプルとしてipmvプロジェクト(画面系)を作成しているので、
そちらを参照してください。
